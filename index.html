A New Code


Programming is in danger. The world is controlled by the Empires' bloatware which is (not so secretly) installed on every single device via IoT.

In his quest to find the one true language, Sakis rediscovers Lisp and decides to visit planet Clojure to see if joining the Clojurian Rebellion is the right choice...
